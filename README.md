# Predictive Analytics Lab Exam-2

## Project Overview
This project performs a binary classification task. It includes Exploratory Data Analysis (EDA), a Random Forest classification model, and a visualization of the decision boundary.

## Dataset
- File: `Lab_Exam_binary_classification_dataset.csv`

## Tasks Completed
* **EDA**: Analyzed feature distributions and correlations.
* **Modeling**: Built a Random Forest Classifier.
* **Visualization**: Plotted the decision boundary.
* **Evaluation**: Reported accuracy and classification metrics.
